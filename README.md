# Inventory Forecast App
在庫データをもとに欠品リスクを予測する業務アプリケーション。

## Features
- Laravelによる在庫CRUD
- 機械学習による需要予測（移動平均＋Holt-Winters）
- 発注推奨量・欠品見込み表示
- GitHub上で要件〜設計〜コードを一貫管理
