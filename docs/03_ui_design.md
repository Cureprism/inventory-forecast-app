# 1. 画面一覧
| No | 画面名 | URL | 概要 |
|----|---------|------|------|
| 1 | ログイン画面 | `/login` | 登録済ユーザーがログインする |
| 2 | ダッシュボード（在庫一覧） | `/products` | 商品ごとの在庫・安全在庫・欠品見込みを一覧表示 |
| 3 | 商品登録/編集 | `/products/create` `/products/{id}/edit` | 商品情報の追加・修正 |
| 4 | 入出庫登録 | `/transactions/create` | 商品と区分(IN/OUT)を指定して数量を登録 |
| 5 | 商品詳細 | `/products/{id}` | 入出庫履歴と軽量予測（SMA＋曜日補正）をグラフで表示 |

## 3. 画面要素（例）

### 3.1 在庫一覧
| 要素 | 内容 |
|------|------|
| 検索フォーム | 商品名・SKUで絞り込み |
| テーブル | 商品名／SKU／現在庫／安全在庫／欠品見込み日 |
| ボタン | [詳細を見る]／[新規登録] |

### 3.2 商品詳細
| 要素 | 内容 |
|------|------|
| 基本情報 | 商品名・SKU・安全在庫・リードタイム |
| グラフ | 在庫推移＋予測線（簡易） |
| テーブル | 入出庫履歴（日付・区分・数量） |
| ボタン | [入出庫登録]／[戻る] |
