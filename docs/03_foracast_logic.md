## 在庫予測ロジック

入出庫履歴から出庫傾向を算出し、現在庫がいつ不足するかを予測。
予測は以下の 2 段階で行う。

## 1. 平均出庫量（1日あたり）の算出

基準日（.env に設定があればそれ、なければ当日）を起点に、  
直近 N 日間の「出庫（OUT）」数量を集計し、1日あたりの平均値を求める。

- 期間：基準日を含む過去 N 日間（既定は 14 日）
- 計算式： `平均出庫量 = 期間内の OUT 合計 ÷ N`  
         `小数第3位で丸め、負になる場合は 0 として扱う。`  
※ サンプルデータを用いた予測結果が毎回同じになるよう、基準日は固定。

## 2. 発注推奨日の計算

以下の日数を算出する。

- **安全在庫を割るまでの日数：**  
  現在庫が安全在庫を下回るまでの日数。

- **補充が間に合うための日数：**  
  「この日までに発注しないと補充が間に合わない」日数。

どちらが先に到達するかを比較し、より早い日を発注推奨日とする。
